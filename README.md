## Build and Setup
```sh
  docker compose build
  # enter in a docker container
  docker compose up -d
  docker cp javascript-ast-tutorial:/usr/src/app/node_modules/ ./
  # enter in a docker container
  docker compose run --rm node sh
  # you can type in a command and hit 'Enter' to execute it.
```
